# Mapa Sonic FonoKids

## Objetivo
Crear un mapa educativo simple para probar las actividades fonoaudiologicas del mod Sonic FonoKids.

## Estructura inicial
1. Sala de inicio
2. Sala de silabas iniciales
3. Sala de vocabulario por categorias
4. Sala final de reporte

## Reglas de diseno
- Sin enemigos.
- Sin precipicios.
- Espacios amplios.
- Caminos claros.
- Objetos educativos deben aparecer frente a Sonic.
- El mapa debe servir para ninos pequenos, no para desafio de plataformas.

- fonospritecheck
